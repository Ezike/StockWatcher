# StockWatcher
Android project that consumes data from a websocket. Built with MVVM, Kotlin Flow, Clean architecture. 

## Prerequisite
To build this project, you require:
- Android Studio Dolphin Canary 5
- Gradle 7.4
- Kotlin 1.5.31
- Android Gradle Plugin 7.0.3
